# Cybersecurity Mastery — Addie Lamarr

A local mirror of [Addie Lamarr's Cybersecurity Mastery vault](https://publish.obsidian.md/addielamarr), downloaded from Obsidian Publish and structured for offline use in Obsidian.

## Overview

Addie Lamarr maintains a comprehensive, publicly published Obsidian vault covering cybersecurity concepts, tools, attack techniques, and study resources. This repo is a downloaded snapshot of that vault — all original content and structure belongs to Addie Lamarr.

## Usage

1. Clone the repository:

```bash
git clone https://github.com/darthrevan030/Cybersecurity-Mastery-Addie-Lamarr.git
```

2. Open the folder as a vault in [Obsidian](https://obsidian.md):
   - **Open Obsidian → Open folder as vault → select the cloned directory**

3. Start from the home note: `00 Home MOC.md`

## How This Was Downloaded

The vault was scraped from the published site using a custom Python tool that extracts Obsidian Publish's internal cache manifest and bulk-downloads all files while preserving the original folder structure.

→ [Obsidian Vault Downloader](https://github.com/darthrevan030/Obsidian-Vault-Downloader)

```bash
python main.py https://publish.obsidian.md/addielamarr ./cybersecurity-vault
```
